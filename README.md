# Mini Project NTT DATA

Membuat aplikasi sederhana menggunakan React 19. Data dummy menggunakan dari [dummyjson](https://dummyjson.com/).

## Fitur

- Login
- Home
- List Product & Search
- Detail Product
- Add Product
- Edit Product
- Delete Product
- Logout

## Shared Component

- Button
- Input
- TextArea
- Select
- Loading

## State Management

Menggunakan Zustand untuk data auth.

## Tech Stack

- Vite + React 19
- React Router
- Zustand
- Typescript
- Tailwind CSS

## Setup

```bash
npm install
npm run dev
```
